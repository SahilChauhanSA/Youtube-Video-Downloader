# YouTube Video Downloader (Python + Tkinter + yt-dlp)

This is a GUI-based YouTube video downloader built using:
- Python
- Tkinter
- yt-dlp
- ffmpeg (for merging audio + video)

## Features
✔ Search available video resolutions  
✔ Download with chosen resolution  
✔ Shows live download progress  
✔ Supports high-quality video (1080p, 4K depending on YouTube availability)

## How to Run
1. Install dependencies:
   pip install yt-dlp
   pip install pillow

2. Install ffmpeg and add it to PATH or set ffmpeg_location in code.

3. Run the project:
   python main.py
